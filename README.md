# vkudahter 
For download images from VK profiles

# Getting started
- Clone the repository
```
git clone https://github.com/ndiuky/vkudahter.git
```
- Install dependencies
```
cd vkudahter
pnpm install
```
- Set environment
```
mv .env.example .env
nano .env
```
- Build and run the project
```
npm start
```
